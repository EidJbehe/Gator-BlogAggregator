# Gator - RSS Feed Aggregator CLI

Gator is a command-line RSS feed aggregator built with **TypeScript** and **PostgreSQL**.  
It allows users to subscribe to RSS feeds, fetch posts automatically, and browse them directly from the terminal.

This project was built as part of the **Boot.dev Backend Development curriculum**.

---

# Features

- User registration and login
- Add RSS feeds
- Follow and unfollow feeds
- Fetch posts from feeds automatically
- Store posts in PostgreSQL
- Browse posts directly from the CLI
- Feed aggregation loop

---

# Tech Stack

- TypeScript
- Node.js
- PostgreSQL
- Drizzle ORM
- fast-xml-parser

---

# Requirements

Before running the program, make sure you have:

- Node.js (v18 or higher recommended)
- PostgreSQL
- npm
